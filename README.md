**#About**  
This is my personal portfolio website made with **HTML** and **CSS**.  
It showcases my skills, projects, and contact information.  

This project was developed as part of the **CSFCP (Computer Science Fundamentals and Career Pathways)** assignment, using **Visual Studio Code, Git, and GitHub** to practice version control and professional workflow.



**#Tools & Commands Used**  
 **Visual Studio Code** – for writing and editing code  
 **Git** – for version control  
   _git init_ : initialize the repository  
   _git add ._ : stage changes  
   _git commit -m "message"_ : save snapshot of changes  
   _git branch -M main_ : set main branch  
   _git remote add origin repo url_ : connect to GitHub  
   _git push -u origin main_ : upload files to GitHub  
**GitHub** – to host the project online and track commits
